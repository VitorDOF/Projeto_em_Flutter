#  AJUDA 192 - Sistema de Atendimento Móvel de Urgência

[![Flutter](https://img.shields.io/badge/Flutter-3.16+-blue.svg)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-3.2+-blue.svg)](https://dart.dev)

Aplicativo mobile desenvolvido em Flutter para facilitar o acesso aos serviços de emergência e saúde, inspirado no SAMU (Serviço de Atendimento Móvel de Urgência).

## Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

- [Git](https://git-scm.com)
- [Flutter](https://flutter.dev/docs/get-started/install) (versão 3.16 ou superior)
- [Android Studio](https://developer.android.com/studio) / [VS Code](https://code.visualstudio.com/)
- Dispositivo ou emulador para testes

## Como executar o projeto

### 1. Clone este repositório
```bash
git clone https://github.com/VitorDOF/Projeto_em_Flutter.git
```
### Acesse
```bash
cd Projeto_em_Flutter
```
### Instale as dependências
```bash
flutter pub get
```
### Execute o aplicativo
```bash
flutter run
