# 🚑 AJUDA 192 - Sistema de Atendimento Móvel de Urgência

[![Flutter](https://img.shields.io/badge/Flutter-3.16+-blue.svg)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-3.2+-blue.svg)](https://dart.dev)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Aplicativo mobile desenvolvido em Flutter para facilitar o acesso aos serviços de emergência e saúde, inspirado no SAMU (Serviço de Atendimento Móvel de Urgência).

### 🔐 Tela de Login
- ✅ Design moderno com gradiente azul
- ✅ Campos de email e senha estilizados
- ✅ Botão de login principal
- ✅ Autenticação com Google
- ✅ Link para recuperação de senha
- ✅ Opção de criar nova conta
- ✅ Layout responsivo para diferentes tamanhos de tela

### 🏠 Tela Home
- ✅ AppBar personalizada com ícones de notificação e perfil
- ✅ Header de boas-vindas com nome do usuário
- ✅ Card de localização atual
- ✅ **Botão de emergência 192** (ligação rápida)
- ✅ **Grid de serviços rápidos** (6 serviços essenciais)
  - Hospitais
  - Primeiros Socorros
  - Ambulâncias
  - Farmácias
  - Contatos Úteis
  - Dicas de Saúde
- ✅ **Lista de hospitais próximos** com informações de distância e horário
- ✅ **Dicas de saúde** com rolagem horizontal
- ✅ Bottom Navigation Bar para navegação entre seções

## 🛠️ Tecnologias Utilizadas

- **Flutter** - Framework UI
- **Dart** - Linguagem de programação
- **Material Design 3** - Design system
- **Google Fonts** - Tipografia personalizada
- **Icons** - Ícones do Material Design
